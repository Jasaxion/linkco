aw.takeoff() - takes off the drone.
aw.land() - lands the drone.
aw.move_left(x) - Fly x cm left.
aw.move_right(x) - Fly x cm right.
aw.move_forward(x) - Fly x cm forward.
aw.move_back(x) - Fly x cm backwards.
aw.flip_left(x) - Flip to the left x time.
aw.flip_right(x) - Flip to the right x time.
aw.flip_forward(x) - Flip forward x time.
aw.flip_back(x) - Flip backwards x time.
aw.rotate_clockwise(x) - Rotate x degree clockwise.
aw.rotate_counter_clockwise(x) - Rotate x degree counter-clockwise.
aw.go_xyz_speed(x, y, z, speed) - Fly to x y z relative to the current position. Speed defines the traveling speed in cm/s.
aw.rotate(x, y) - Rotate first drone x degree clockwise, rotate second drone y degree clockwise.
aw.face_takeoff() - Take off for tracking face.
aw.track_face() - Track face.
